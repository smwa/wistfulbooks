About Librivox Libre

Welcome to Librivox Libri! This site takes the great work that Librivox.org has done and packages it into a single page that lets you listen to audiobooks in your browser. You can download books so that you can come back here without internet, no matter where you are and whether you are on your phone, tablet, or computer.

This site is provided for free, and the source code can be found at https://github.com/smwa/librivoxlibri. Feel free to create an issue or pull request there if you have anything you would like to change.

The following paragraph is not currently accurate. Librivox Libri uses ipfs.io to distribute the audiobooks without being dependent on the original website being up. By using this site, you agree to help the greater community by contributing your resources, such as your bandwidth and storage, to send any audiobooks and content needed for Librivox Libri that you have downloaded to others that are interested in using the site.

If you enjoy the service and would like to help make more books available to everyone, you can click "Make Available Offline" for books above that interest you. They will be downloaded and then shared with the next person that is interested in it.
